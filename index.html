◯1 검색내용 : 주소를 입력하고 검색 버튼을 누르면 위치로 이동
   주소 검색에 실패하면 키워드를 검색
◯2 키워드 검색 결과 : ◯1에 입력된 키워드를 검색하여 리스트 항목으로 표시
   항목을 클릭하면 위치로 이동(마커와 마커를 누르면 내용창 표시)
◯3 엑셀파일 선택버튼 : 파일 선택 창을 통하여 엑셀 파일을 선택(10개 행까지 표현)
  예시로 사용된 엑셀데이터(7개 행[A:G] 사용)
◯4 주소행 선택 : 엑셀에 입력된 필드 중 주소 내용이 들어 있는 행을 선택
◯5 엑셀 데이터 검색 키워드 : 불러온 엑셀 내용 중 특정 내용만 보고 싶은 경우 검색 단어 입력
◯6 검색버튼 : 검색 키워드로 검색하여 검색된 내용만 리스트로 표시
◯7 해제버튼 : 모든 엑셀데이터 리스트에 표시
◯8 모두표시 : 모든 엑셀 주소 위치에 마커를 표시하고 마커를 누르면 엑셀데이터가 표시된 
윈도우가 생김, 윈도우는 x버튼을 눌러 지울수 있음
◯9 엑셀데이터 항목 선택 : 선택된 항목 주소로 이동하고 마커를 표시하고 마커를 누르면 
엑셀데이터가 표시된 윈도우가 생김
◯10 에러내용 : 주소검색에 실패하면 에러 내용 표시
◯11 분할경계선 : 분할 경계선에 마우스를 올려서 마우스가 화살표로 바뀐 상태에서 드래하여 조정
